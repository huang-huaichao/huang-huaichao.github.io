# 大纲总览

> 《机器学习在算法交易中的应用》（第二版）—— Stefan Jansen

本书系统介绍了从数据工程到深度强化学习的完整机器学习交易工作流。全书共四大部分：

## 第一部分：数据，阿尔法因子与投资组合

市场数据与基本面数据的获取处理、金融另类数据挖掘、Alpha 因子设计与研究、投资组合优化与绩效评估。

## 第二部分：机器学习在交易中的基本应用

机器学习建模全流程、线性模型（回归与正则化）、时间序列模型（ARIMA、VAR）、随机森林、Boosting 算法（XGBoost、LightGBM 等）、贝叶斯机器学习、ML4T 端到端策略回测。

## 第三部分：自然语言处理

文本数据与情感分析、主题建模（LDA）、词向量表示（Word2Vec、Doc2Vec）。

## 第四部分：深度学习与强化学习

深度学习基础（TensorFlow / PyTorch）、RNN/LSTM 时间序列建模、CNN 处理图像与文本、自编码器特征提取与风险检测、GAN 合成金融数据、深度强化学习构建交易代理。

## 写在最后

个人总结与体悟。
