# 博客项目测试文档

> 本文档用于测试博客的各项功能特性

## 项目概述

**博客地址**: https://learnergalaxy.github.io

**技术栈**:
- Hexo (静态博客生成器)
- NexT 主题 (Mist 风格)
- GitHub Pages (部署平台)

**创建时间**: 2025-02-25

- Git: 2.50.1

**项目位置**: `~/my-blog`

## 常用命令

### 创建新文章
```bash
cd ~/my-blog
hexo new "文章标题"
```
新文章会生成在 `source/_posts/文章标题.md`

### 本地预览
```bash
cd ~/my-blog
hexo clean && hexo g && hexo server
```
访问 http://localhost:4000
停止: 按 `Ctrl+C`

### 部署到 GitHub Pages
```bash
cd ~/my-blog
hexo clean && hexo g && hexo d
```

### 清理缓存
```bash
cd ~/my-blog
hexo clean
```

13. ⏱️ **网站运行时间统计**: 页脚显示运行时间

## 已安装的插件

### 1. hexo-generator-searchdb
**用途**: 本地搜索功能
```bash
npm install hexo-generator-searchdb --save
```

### 2. hexo-symbols-count-time
**用途**: 文章字数统计和阅读时间
```bash
npm install hexo-symbols-count-time --save
```

### 3. hexo-generator-index
**用途**: 生成首页索引（Hexo 内置）

### 4. MathJax
**用途**: 数学公式渲染
- 配置在 `_config.yml` 中
- 支持行内公式 `$...$` 和块级公式 `$$...$$`
